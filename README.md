# AirBnTruta-Web

É um site que simula uma plataforma de hospedagens nada convencional: aqui, fugitivos procuram abrigo e hospedeiros oferecem seus espaços.
Na página inicial, o visitante encontra hospedagens disponíveis e pode filtrar por localidade ou preço.
Cada hospedagem tem uma descrição detalhada e abre espaço para que o fugitivo faça sua proposta de estadia.
O fugitivo acompanha seus interesses em uma área própria, enquanto o hospedeiro recebe notificações e decide quem aceitar.
O projeto foi desenvolvido com Spring Web, Thymeleaf e JDBC como parte da disciplina, e apresentado em vídeo.
